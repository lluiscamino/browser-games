# Browser Games

A collection of HTML games to play solo or with friends!

Play them online at [lluiscamino.me/browser-games](https://lluiscamino.me/browser-games/)

---

## Development

This project uses [Vite](https://vite.dev/) as the build tool.

### Quick Start

1. **Install dependencies:** `npm install`
2. **Start dev server:** `npm run dev`
3. **Build for production:** `npm run build`

> For advanced CLI usage, refer to the [Vite Documentation](https://vite.dev/guide/cli) or check the scripts in [
`package.json`](package.json).
